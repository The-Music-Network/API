# API
This is the backend API for The-Music-Network

## Endpoints
| Method | URI           | Controller.Action      |
| ------ |:-------------:| ----------------------:|
| GET    | /posts        | PostController.index   |
| GET    | /posts/{id}   | PostController.show    |
| GET    | /users        | UserController.index   |
| GET    | /users/{id}   | UserController.show    |
